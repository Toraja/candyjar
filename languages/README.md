# Langeuages

## Requirements

Also, each language might have additional requirements. Refer to `README` for each language.

- [Jupyter Notebook](https://jupyter.org/)
  - Some snippets are written as jupyter notebook.
  - To install:
    ```sh
    pipx install jupyterlab
    ```
- [Jupytext](https://github.com/mwouts/jupytext)
  - [Documentation](https://jupytext.org/)
  - Convert between `ipynb` <-> markdown and many others.
  - Each code block is put into code cell and remainings are treated as markdown.
  - To install:
    ```sh
    pipx install jupytext
    ```
- [Jupyter Real-Time Collaboration](https://github.com/jupyterlab/jupyter-collaboration)
  - This is technically an optional, but it enables autoloading of modified files on the disk and it makes a lot easier to work on the notebooks generated by jupytext.  
    (Do not use `pipx`)
    ```sh
    pip install jupyter-collaboration
    ```
